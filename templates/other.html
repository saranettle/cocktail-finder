Cocktail Finder is the culmination of the web development skills I have learned over the course of two semesters. The data seen on this page was gathered with a Python web scraper from the Wikipedia page for the International Bartending Associationâ€™s list of official cocktails. This cocktail scraper (see the github repo here) collected the name, primary alcohol, and the ingredients and then saved this information to a .csv file. Afterwards, I took this .csv file and converted it into a MySQL database. Finally, I used the Flask Python framework to create this website and dynamically generate pages for each drink. Creating this web app was challenging on all levels, but I am excited to share it with everyone. Please check out the github repo to learn more!
